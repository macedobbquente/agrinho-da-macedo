# Projeto Agronegócio Brasileiro

## Descrição

Este projeto apresenta informações básicas sobre o agronegócio brasileiro, destacando sua importância econômica, principais produtos e curiosidades.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Apresentação do conceito de agronegócio
- Lista dos principais produtos agrícolas
- Informações sobre a importância econômica do setor
- Botão interativo com curiosidades aleatórias

## Estrutura

```text
agronegocio/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Como Executar

1. Baixe os arquivos.
2. Coloque todos na mesma pasta.
3. Abra o arquivo `index.html` no navegador.
